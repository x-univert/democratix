## 2. Présentation du Projet

### 2.1 Genèse

Le projet DEMOCRATIX naît du constat de l'inefficacité des systèmes de vote actuels :
- Coûts prohibitifs (200M€ pour une présidentielle)
- Manque de transparence
- Abstention croissante (30-40% en moyenne)
- Fraudes régulières dans certaines démocraties

### 2.2 La Solution Technique

Plateforme de vote décentralisée combinant :

**Blockchain MultiversX**
- 100,000 TPS
- Finalité 6 secondes
- Coût 0.001€/transaction

**Cryptographie avancée**
- zk-SNARKs (anonymat)
- Chiffrement homomorphique (comptage)
- Blind signatures (non-traçabilité)

**Infrastructure moderne**
- Backend Rust/Node.js
- Frontend React
- Mobile iOS/Android
- API publique

### 2.3 Différenciation

| Critère | DEMOCRATIX | Concurrents |
|---------|------------|-------------|
| Décentralisation | ✅ Publique | ❌ Privée |
| Open Source | ✅ Oui | ❌ Non |
| Coût | ✅ 0.10€/vote | ❌ 1-5€/vote |
| Conformité EU | ✅ RGPD, eIDAS | ⚠️ Partielle |
| Cryptographie | ✅ zk-SNARKs | ❌ Basic |
| Scalabilité | ✅ 10M votes/h | ❌ <1M votes/h |

### 4.3 Modèles de Revenus

#### Modèle SaaS (Software as a Service)

**Pour Collectivités** :
```
Prix = Forfait de base + (Nb électeurs × Tarif unitaire)

Forfait : 500€
Tarif unitaire : 0.10€/électeur/an
```

**Exemple** :
- Commune 10k habitants : 500€ + (10k × 0.10€) = 1,500€/an
- Département 500k habitants : 500€ + (500k × 0.08€) = 40,500€/an

**Pour Entreprises** :
```
Prix = Forfait + (Nb votants × 0.20€)

Forfait : 200€/vote
```

**Exemple** :
- PME 200 salariés (AG) : 200€ + (200 × 0.20€) = 240€/vote

## 8. Annexes

### 8.1 Étude de Cas : Estonie

L'Estonie a implémenté le vote électronique (non-blockchain) en 2005 :

**Résultats** :
- 46% d'adoption (2019)
- Économies : 11,000 jours de travail/élection
- Participation jeunes : +15%
- Coût : 2€/vote (vs 5-10€ traditionnel)

**Leçons** :
- Adoption progressive sur 15 ans
- Confiance = éducation + transparence
- Support politique bipartisan essentiel

**Notre avantage** :
- Technologie supérieure (blockchain vs centralisé)
- Coût 20x inférieur (0.10€ vs 2€)
- Open source (vs propriétaire estonien)
